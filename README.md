# AttrJson::Associations

An association extension for attr_json.

## Dependencies

* ruby 2.7
* attr_json 1.2
* activerecord 6.0
* activesupport 6.0

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'attr_json-associations'
```

Then execute:

    $ bundle
